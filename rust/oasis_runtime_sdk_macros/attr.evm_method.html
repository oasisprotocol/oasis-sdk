Attribute Macro evm_method 

Source 
#[evm_method]
Expand description

A helper attribute for #[sdk_derive(...)]. It doesn’t do anything on its own; it only marks functions that represent contract methods.

The permitted forms are:

  • #[evm_method(signature = "...")]: The method selector is computed from a Solidity method signature, and the method takes the precompute handle and data offset as parameters.
  • #[evm_method(signature = "...", convert)]: The method selector is computed from the signature, the arguments are automatically decoded and passed to the marked method, which must have the appropriate number and type of arguments.
