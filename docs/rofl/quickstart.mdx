# Quickstart

## Prerequisites

### Containerized App

Application orchestrated with Docker or Podman compose.
 - App images available to download on a public URL.

### Oasis CLI

Download the latest release of the Oasis CLI [here][oasis-cli-dl] and
install it on your computer.

[oasis-cli-dl]: https://github.com/oasisprotocol/cli/releases

### Some ROSE to rent the machine and pay blockchain fees.

If you don't have an existing account on Oasis Sapphire Testnet chain, you will
need to create or import it with the [`oasis wallet create`] or [`oasis wallet
import`] commands respectively. Then, fund the acount with ~110 TEST tokens: 100
tokens for the ROFL registration escrow and another 10 TEST or so for paying
different kinds of gas fees. To fund the account with TEST tokens, visit the
official [Oasis Testnet faucet] or reach out to us on the
[`#dev-central` channel on Discord][discord].

[`oasis wallet create`]: https://github.com/oasisprotocol/cli/blob/master/docs/wallet.md#create
[`oasis wallet import`]: https://github.com/oasisprotocol/cli/blob/master/docs/wallet.md#import
[Oasis Testnet faucet]: https://faucet.testnet.oasis.io
[discord]: https://oasis.io/discord

## Roflize your App

We will build our application ROFL in five steps:

1. [Initialize](#initialize) the ROFL manifest.
2. [Create](#create) a new app on blockchain.
3. [Build](#build) a ROFL bundle.
4. Encrypt and safely store [Secrets](#secrets).
5. [Deploy](#deploy) an app to the ROFL node.

### Initialize

Inside our project folder, run [`oasis rofl init`] to generate the
initial `rofl.yaml` manifest file and to register a new ROFL app on Sapphire
Testnet:

```shell
oasis rofl init
```

### Create

Create a new ROFL app on Sapphire with the [`oasis rofl create`] command. Use
`--network` parameter to specify Testnet or Mainnet chain.

```shell
oasis rofl create --network testnet
```

You should be able to find your ROFL app now on the [Oasis Explorer].

### Build

Next, build the *ROFL bundle*.

<Tabs>
    <TabItem value="Native Linux">
        ```shell
        oasis rofl build
        ```
    </TabItem>
    <TabItem value="Docker (Windows, Mac, Linux)">
        ```shell
        docker run --platform linux/amd64 --volume .:/src -it ghcr.io/oasisprotocol/rofl-dev:main oasis rofl build
        ```
    </TabItem>
</Tabs>

### Secrets

If your application uses environment variables you would like to privately
store on-chain, use the [`oasis rofl secret`] command:

```shell
echo -n "$TOKEN" | oasis rofl secret set TOKEN -
```

To submit this secrets and the hash of previously compiled ROFL bundle, run
[`oasis rofl update`]:

```shell
oasis rofl update
```

Secrets names and the new ROFL bundle metadata should be visible now on the
[Oasis Explorer].

### Deploy

Deploy your app to the ROFL provider with the [`oasis rofl deploy`] command:

```shell
oasis rofl deploy
```

The Oasis CLI will automatically pick the offer that fits the required
resources.

Congratulations, you have just deployed your first ROFL app! 🎉

You can check the status of the rented machine with [`oasis rofl machine
show`]:

```shell
oasis rofl machine show
```

To fetch your application logs, use [`oasis rofl machine logs`]:

```shell
oasis rofl machine logs
```
